# 📌 Overview

This project is a browser-based Tic Tac Toe game where two players play against each other using X and O symbols.

The game includes:
- Turn switching between players
- Winner detection
- Draw detection
- Reset game functionality
- Interactive and clean user interface

This project was created to practice front-end web development fundamentals such as:
- DOM Manipulation
- Event Listeners
- Arrays and Loops
- Functions
- Conditional Statements
- CSS Grid Layout
- Game Logic

# 🎯 How to Play

1. Open the game in your browser.
2. Player X starts first.
3. Click on any empty square.
4. Players alternate turns between X and O.
5. The first player to match 3 symbols in a row wins.
6. If all squares are filled without a winner, the game ends in a draw.
7. Press the Reset button to restart the game.

---

# 🧠 Game Logic

The game works by:
- Storing player moves inside an array
- Checking winning combinations after every move
- Preventing players from selecting occupied cells
- Detecting draws when the board becomes full
- Resetting all values when restarting the game
